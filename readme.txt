== Description ==

A set and forget solution for WordPress speed.

*[Test your site speed](https://gtmetrix.com/) before and after.*

**Features**

* Add better cache handling
* Add expires headers

Ask for help: [help@veryeasy.io](mailto:help@veryeasy.io)

== Installation ==

**Automatic**

* From your WordPress Admin, navigate to: **Plugins** > **Add New**
* Search for: **Speed Made Easy**
* Install it
* Activate it

**Manual**

* Download
* Unzip
* Upload to /plugins/ folder
* Activate
